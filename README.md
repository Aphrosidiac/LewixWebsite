# LEWIX.AI

Flagship website for [Lewix.ai](https://lewix.ai) — a software engineering studio based in Johor Bahru, Malaysia.

## Stack

- **Next.js 16** (App Router, React 19, TypeScript)
- **Tailwind CSS v4**
- **GSAP + ScrollTrigger** (scroll-driven animations)
- **Lenis** (smooth scrolling)
- **Motion v12** (page transitions, component animations)

## Design

Dark theme (`#050505`) with muted indigo (`#818CF8`) accent. Typographic, architectural layout inspired by Mattis Studio. Glassmorphism navbar with morphing glass bubble. Dot grid + gradient mesh background. SVG noise grain overlay.

## Pages

- `/` — Home (hero, marquee, manifesto, services, numbers, CTA)
- `/work` — Coming soon
- `/services` — Service breakdowns + process
- `/about` — Company story + principles
- `/contact` — Form + WhatsApp contacts

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deploy

Target: Vercel (lewix.ai domain)

```bash
npm run build
```
